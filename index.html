ogni esecuzione del blocco si chiama iterazione

l'istruzione break

Serve ad uscire dal ciclo prematuramente


							i = 1
							while i < 6:
    						   print(i)
    						   if i == 3:
        					      break
    						   i += 1
							print("fine loop")

l'istruzione continue

serve a saltare l'iterazione corrente ed eseguire quella successiva


							i = 0
							while i < 6:
							  i += 1
							  if i == 3:
							    continue
							  print(i)

Gli array (vettori)

Un array è una variabile che può contenere più valori.

Si immagini di avere la seguente situazione


							car1 = "Ford"
							car2 = "Volvo"
							car3 = "BMW"
  • Per aggiungere un nuovo valore (Tesla) bisogna creare una nuova variabile.
  • Per aggiungere 10000 nuovi valori, bisogna creare 10000 nuove variabili.
  • Tra queste è poi difficile trovare quella che ha un certo valore.

In tali situazioni tornano utili gli array.

Sintassi


							cars = ["Ford", "Volvo", "BMW"]

Accedere agli elementi dell'array


							x = cars[0]
  • Ogni elemento ha una posizione.
  • Il primo ha posizione 0, il secondo 1, etc.

Modificare il valore di un elemento


							cars[0] = "Toyota"

Lunghezza di un array


							x = len(cars)

Slicing operation

nomeArray[ inizio : fine : step ]


							mioArray = [1, 2, 3, 4, 5, 6, 7]
							print(mioArray[1:4:2])

Impostare step -1 per ottenere gli elementi al contrario.

Questa sintassi funziona anche con le stringhe

Il ciclo for

Con il ciclo for possiamo eseguire un blocco di istruzioni per ogni elemento dell'array


							fruits = ["apple", "banana", "cherry"]
							for x in fruits:
  							  print(x)

Se vogliamo il numero dell'iterazione corrente possiamo adottare diverse strategie.

Una strategia non immediata ma sintatticamente elegante è la seguente: 


							mioArray=[1, 2, 3, 4, 5, 6, 7]
							for x in range(len(mioArray)):
    						    print("l'elemento in posizione ", x, "è: ",mioArray[x])

una espressione è una concatenazione di operazioni e funzioni

Nested loops


							cars = ["BMW", "Jaguar", "Alfa-Romeo"]
							versions = ["Berlina", "SUV", "Cabrio", "Coupe"]
							
							for car in cars:
								print(car, " Versions: ")
								for version in versions:
									print(version, end="; ")
								print("\n\n")

Funzioni

  • Una funzione è un blocco di codice che viene eseguito quando lo invochiamo
  • Gli possiamo passare dei dati in input (parametri)
  • Possiamo ottenere un valore in output (return)

Si definisce con def


							def my_function():
								print("Hello from a function")

Si chiama scrivendo il suo nome seguito da parentesi


							my_function()

Passiamo i parametri in input, specificandoli tra parentesi.


							def stampa_nome(nome):
								print("Ti chiami ", nome, " Rossi")
							
							stampa_nome("Damiano")
							stampa_nome("Valentino")

Possiamo specificare più parametri, separandoli con la virgola


							def stampa_nome(nome, cognome):
								print("Ti chiami ", nome, " ", cognome)
							
							stampa_nome("Damiano", "Stopo")
							stampa_nome("Valentino", "Rossi")
